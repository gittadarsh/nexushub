NexusHub
One place to see every club event on campus - and find a team for the ones that need one.
     🔗 Live: nexushub-five-phi.vercel.app
     
The problem
College clubs run entirely on Instagram posts and WhatsApp groups. That means:

Students miss events that were only announced in class or on a story that disappeared.
Team-based events exclude students who don't already have a friend group to join with.
Registration is repetitive - the same name, roll number, and details typed into a new Google Form every single time.
Clubs get no real data - no clean registrant list, no easy way to show a sponsor "this many people actually showed up.         

NexusHub doesn't try to replace Instagram or WhatsApp - clubs keep posting there exactly as they do now. It's the layer neither of them actually covers: structured registration, team-matching, and guaranteed visibility that isn't dependent on an algorithm.

Who it's for
Students - one feed for every club, register in a few taps, find teammates for events that need them, ask a club a direct question and actually know when they've replied.
Clubs - post once using the same poster they already made for Instagram, get a clean approve/reject queue for payments instead of chasing screenshots, and hand out the WhatsApp group link only to people who've actually paid.
